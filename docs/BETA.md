# Beta（测试版）说明

DocsFabrica 当前版本为 Beta（测试版），主要用于功能测试和用户反馈收集。

测试版本中的功能、界面、交互方式及软件行为可能会随着后续版本迭代而调整，恕不另行通知。

由于测试版本仍处于持续开发阶段，软件的稳定性、兼容性及运行表现可能与正式版本存在差异，个别功能可能出现异常或未完全实现。

建议用户在处理重要文档前做好备份，并优先使用文档副本进行测试。对于测试版本可能导致的数据丢失、文档异常或其他不可预期的问题，开发者将持续修复和改进，但不保证所有问题均能及时解决。

欢迎用户通过 GitHub Issues 提交 Bug 报告、功能建议及使用反馈，以帮助我们不断完善 DocsFabrica。
